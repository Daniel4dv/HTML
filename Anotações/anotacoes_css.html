Como estilizar

Estilos inline:

Aplicando estilos dentro da tag:

<h1 style="parametro desejado" > "titulo" </h1 >

A parte destacada em vermelho demonstra como e onde deve ser usado o parametro de estilo

"color:red;"
"font-size:20px"
"font-family:arial"

CSS no head:

É possivel adicionar os estilos dentro do cabeçalho através da tag: <style>

<head >
<style>
body{
"parametros desejados"
}
h1{
"parametros desejados"
} </style > </head> <body> </body>

CSS Externos:

É possivel linkar arquivos externos(CSS) em qualquer codigos HTML através do comando:

<link rel="stylesheet" href= "nome_do_arquivo_css" >

Tudo que for escrito nesse arquivo, será implementado automaticamente na pagina

Dica:

Ao segurar "ctrl" e clical em um arquivo que não exista, você consegue cria-lo automaticamente

Familia de fontes

Você pode alternar entre fontes com o comando : font-family: exemplo

Exemplo

        h1 {
    font-familly: arial;
        }

É interessante usar mais de uma fonte para atender a todos os dispositivos


Tamanho de fontes

Alternando tamanho do texto com : font-size: tamanho

É recomendado o uso de px(pixels) ou em(relativo ao tamanho original(16px)) na definição do tamanho das fontes

Exemplo:


    h1
    {
        font-size:16px
    }

peso das fontes

Alternando o peso(grossura) da fonte com: font-weight

Exemplo

font-weight:lighter

Pesos possíveis: Lighter,normal,bold,bolder. 100~900

Simplificando o estilo de fonte

É possivel escrever dessa forma mais prática:

ordem: font-style > font-weight > font-size > font-familly

font: italic    bold     2em     'work Sans',sans-serif;

Usando fontes externas

Google fonts

Para usar o google fonts você deve copiar o link do @import do site e colar ele no doc CSS

Fontes baixadas

Para importar uma fonte baixada voce importa ela através do comando:


    @font-face {
        font-family: azonix (nome da fonte) ;
        src: url(azonix.otf) format('opentype'); (nome do arquivo e formato)
                }

Seletores personalizados

ID

id="nome do id" dentro da tag


        <p> <a id="voltar" href="#topo">Voltar ao topo</a></p>

"#topo" é referente a id criada anteriormente, nos referimos a uma id com #seuNome

classes

É recomendado ser usado toda vevz que for necessário aplicar em mais de uma tag

Funciona da mesma forma do que o #ID

class="nome da class" dentro da tag

    
        <li class="blocos">serif</li>

Também é possível adicionar mais de uma classe na mesma tag além de uma ID em uma tag com classe

    <h1 class="principal moderno"> título</h1>
    
    <h1 id="principal" class="moderno"> título</h1>
